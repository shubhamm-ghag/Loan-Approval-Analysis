# Loan-Approval-Dashboard

1. ( HEADLINE )
Loan Approval Analysis = This project is all about the end-to-end data analyst for analyze loan approval trends.

2. ( PURPOSE )
= This project covers the full data pipeline—from cleaning "dirty" raw data to building an interactive business dashboard.
= I analyzed a dataset of loan applications to find insights into approval rates, city-wise trends, and high-value customer segments.

3. ( TECH STACK )
* Excel: Initial data cleaning, blank cell identification, and duplicate removal.
* SQL (PostgreSQL): Deep cleaning, standardizing inconsistent data, and analysis Queries.
* Power BI: Data visualization and dashboard formatting.

4. ( Data Source )
Source = Google. 

5. ( BUSINESS INSIDE )
= Total Volume: Processed 28 clean loan applications.
= Financial Averages: The average loan amount is approximately 62,000 with an average credit score of 700.
= Segment Analysis: Identified key trends in city-wise applications and high-value customer clusters.
